This code allows you to parse information about vacancies on hh.ru
It scans the home page, finds vacancies, follows links to vacancies and collects the specified data

Import and settings

The main libraries for parsing are:
1. requests- makes a request to the site and receives data
2. bs4(BeautifulSoup) - parses the result of the request requests

Scanning the job vacancy

This function extracts the necessary data from the page

def visit_and_check(url, soup, response):
1  global df

2  fkeys = []

3  for keyword in keywords:
4      if keyword in response.text:
5        fkeys.append(keyword)
  # Finding tags (skills) at the bottom of the page
6  skill_elements = soup.find_all('li', {'data-qa': 'skills-element'})
  skills = [li.find('div', class_=re.compile(r'magritte-tag__label')).text for li in skill_elements]

7  header = soup.find('div', {'class': 'vacancy-title'})
8  title = header.find('h1').text

9  salary_str = header.find('span', {'data-qa': 'vacancy-salary-compensation-type-net'})
  if salary_str:
    # Find the first number in a string using a regular expression
      match = re.search(r'\d+', salary_str.text.replace('\xa0', ''))
      if match:
        salary = int(match.group())
      else:
        salary = None
  else:
    salary = None

10  df = df.vstack(pl.DataFrame({
    "URL": [url],
    "Вакансия" : [title],
    "Зарплата" : [salary],
    "Keyword": [', '.join(fkeys)],
    "Tags": [', '.join(skills)]}))

Page navigation and link search

This loop scans the main pages (where job vacancies are located), depending on the iteration, it navigates through pages 2, 3, 4… on the website

1for p in range(9):
    if p > 0:
2        url = f'{start_url}&page={p}'
        print(f"--- Page {p} --- --- ---")

    # Searching for all links inside h2 tags
3    for i in range(9):
4        time.sleep(3)
        print(f"--- --- --- Iteration {i}")
5        response = requests.get(url, headers=headers)
6        soup = BeautifulSoup(response.text, 'html.parser')
7        if soup.find_all('h2'):
            for h2 in soup.find_all('h2'):
                for a in h2.find_all('a', href=True):
                    absolute_url = requests.compat.urljoin(url, a['href'])
8                    check_correctly(absolute_url)
9            break
